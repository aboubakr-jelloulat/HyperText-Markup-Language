HTML Entities, Emojis, and Math Symbols

This document demonstrates various HTML entities, emojis, and math symbols:

Display HTML tag as it is: b tag = <b>.

Common HTML Entities: Less than (&lt;), Greater than (&gt;), Ampersand (&amp;), Non-breaking Space (&nbsp;), Copyright (&copy;), Registered Trademark (&reg;), Euro Sign (&euro;).

Mathematical Symbols: Plus/Minus (&plusmn;), Multiplication (&times;), Division (&divide;), Square Root (&radic;), Pi (&pi;), Integral (&int;), Almost Equal to (&asymp;).

Emojis: Smiley Face (😀), Thinking Face (🤔), Heart (❤️), Star (⭐).

Arrows: Right Arrow (&rarr;), Left Arrow (&larr;), Up Arrow (&uarr;), Down Arrow (&darr;).
